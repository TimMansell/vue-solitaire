# Tests

This folder contains end-to-end tests and fixtures.

## Unit tests

Unit tests are located inside `__tests__`.

Current unit test coverage:

1. [App](/src)
1. [Components](/src/components)
1. [Service](/src/services/solitaire)

## Running tests

### Run unit tests

```
yarn test:unit
```

### Run end-to-end tests

```
yarn test:e2e
```
